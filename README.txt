# Mobile Legends Game

## Description
A basic project structure integrating various languages and technologies for a Mobile Legends-themed game.

## How to Run
1. For Python: Run `app.py` with Flask.
2. For Ruby: Run `server.rb` with Sinatra.
3. For C++: Compile and run `main.cpp`.
4. For Java: Compile and run `Main.java`.
5. For PHP: Serve `index.php` with a PHP server.
6. For SCSS: Compile `style.scss` to `style.css`.

## Note
Ensure you have the appropriate environment set up for each language.
